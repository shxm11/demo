1. Check if Git is initialized
git status

If you get:

fatal: not a git repository

then initialize it:

git init
------------------------------------------------------------
2. Connect your local folder to GitHub
git remote add origin https://github.com/shxm11/demo.git

Verify it:

git remote -v

You should see something like:

origin  https://github.com/shxm11/demo.git (fetch)
origin  https://github.com/shxm11/demo.git (push)
------------------------------------------------------------
3. Create a file

For example:

echo "# Demo Repository" > README.md

Or create any file you want.
------------------------------------------------------------
4. Stage the files
git add .
------------------------------------------------------------
5. Commit
git commit -m "Initial commit"
------------------------------------------------------------
6. Rename the branch to main (if needed)
git branch -M main
------------------------------------------------------------
7. Push to GitHub
git push -u origin main

The -u option sets origin/main as the default upstream, so next time you only need:

git push
